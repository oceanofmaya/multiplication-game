## Getting Started

### Prerequisites
* NodeJS
* Git

### Installing
Fork and clone repository
```
git clone https://github.com/oceanofmaya/multiplication-game.git
```
Navigate to directory
```
cd multiplication-game
```
Get dependencies
```
npm install
```
Run locally
```
node server.js
```
Open http://localhost:3000 in your browser to play
