## Getting Started

### Prerequisites
* NodeJS
* Git

### Installation
Clone repository
```console
$   git clone https://github.com/oceanofmaya/multiplication-game.git
```
Navigate to directory
```console
$   cd multiplication-game
```
Get dependencies
```console
$   npm install
```
Run locally
```console
$   npm run start
```

### Usage
Open http://localhost:3000 in your browser to play
